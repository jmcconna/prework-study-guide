HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use H1 through H6 in order, and only use H1 once per page.
  • Use semantic elements as often as possible for accessibility.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • before opening a pull request, make sure to check for the most updated version of the main branch
  • git flow: create a new branch, add code, commit code, push the code to the branch, then open a pull request, maybe review/test, and merge changes with the main branch.
